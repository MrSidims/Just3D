# Extension Icons

To generate the icons for this extension:

1. Open `create-icons.html` in your web browser
2. Click the "Download All Icons" button
3. Save the downloaded icons to this directory

Alternatively, you can create your own icons in any image editor. Required sizes:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

The icons should represent a 3D model or cube to indicate the extension's purpose.
