## FREQUENTLY ASKED QUESTIONS (for store page)

**Q: Is this extension free?**
A: Yes, Just3D is completely free and open source.

**Q: Do you upload my 3D models?**
A: No, all processing happens locally in your browser. Your files never leave your computer.

**Q: Does it work offline?**
A: Yes, after installation, Just3D works completely offline.

**Q: What's the file size limit?**
A: There's no hard limit, but very large files (100MB+) may affect performance depending on your computer.

**Q: Can I edit 3D models?**
A: Creator Mode allows you to create and modify basic 3D objects. For advanced editing, use dedicated 3D software.

**Q: Are animations supported?**
A: Currently, models display in their default pose. Animation support may come in future updates.

**Q: Does it support textures?**
A: Yes, for formats that include texture data (GLTF, FBX, OBJ+MTL), textures are displayed.

**Q: Can I load models from URLs?**
A: Yes! You can paste any direct link to a 3D model file or right-click on 3D model links and select "Open in Just3D". Works with GitHub, cloud storage, and any publicly accessible URL.

**Q: What permissions does this extension require?**
A: Just3D requires 3 permissions: storage (for preferences), contextMenus (for right-click menu), and host permissions (to fetch URLs you provide). All network requests are user-initiated only - we never fetch data in the background.
